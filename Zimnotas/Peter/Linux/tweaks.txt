**Harde schijven willen password om te mounten**
	HDD : 
		aangepast in etc/fstab :
			1. lsblk -f om drive's UUID te vinden
			2. edit /etc/fstab
				UUID=<drive-uuid> <mount-point> <filesystem-type> <options> 0 2
					UUID = resultaat van lsblk -f
					mount point = directory waar drive moet te vinden zijn
					file system type = bvb btrfs, ext4, ...
					options = ???
					0 2 = ???

	ext SSD's : 	
		nog proberen (eerst rood!)
